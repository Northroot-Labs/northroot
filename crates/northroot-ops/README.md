**Operator and method manifests: schemas, examples, and validators.**

This crate defines operator and method manifests for the Northroot proof algebra system. It provides schemas, examples, and validators for operator definitions and method compositions.

## Purpose

The `northroot-ops` crate provides:

- **Operator manifests**: Schemas and validators for operator definitions
- **Method manifests**: Schemas for method compositions (DAGs of operators)
- **Validation**: Runtime validation of operator and method structures
- **Examples**: Reference implementations and test fixtures
