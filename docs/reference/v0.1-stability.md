# Northroot v0.1 Stability Contract

Status: Release candidate contract

Northroot v0.1 is the first stable release of the verifiable-event kernel. It is
not a runtime, scheduler, policy engine, workspace vault, product workflow, or
governance authority.

## Stable Kernel Surface

The v0.1 stable surface is:

- deterministic canonicalization through `northroot-canonical`;
- content-derived `event_id` computation and verification;
- `.nrj` append-only journal framing through `northroot-journal`;
- public CLI commands for `canonicalize`, `event-id`, `append`, `read`, and
  `verify`;
- offline verification without provider, network, policy, or product
  dependencies.

Breaking changes to canonical bytes, event identity, journal framing, or public
kernel APIs require a later major contract version.

## Incubating Surfaces

Work ledger helpers, portable bundle checks, segmented-journal checkpoint
helpers, state-recovery snapshots, substrate policy refs, and any
workspace/vault/runtime vocabulary remain incubating profiles or consumer
protocols. They may be used for dogfood, but they are not public kernel command
semantics.

The kernel may preserve and verify profile-bearing events. The kernel must not
decide profile meaning.

## Dogfood Gate

The v0.1 release gate requires at least two downstream repos to produce adoption
reports showing repeatable Northroot usage. A passing report must include:

- repo name;
- check name;
- generated or verified journal path;
- total event count;
- kernel-valid event count;
- profile-valid event count;
- invalid event count;
- projection rebuild status.

Northroot imports only adoption reports. It does not import downstream source
data, product state, private policy bundles, or runtime credentials.

## Readiness Command

Run:

```bash
python3 scripts/v01_readiness.py --json
```

The readiness check fails when required docs are missing, release/version drift
is detected, fixture checks fail, current stable CLI commands are not covered by
release checks, or fewer than two passing downstream adoption reports are
available.
